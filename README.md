# Port scaner
This application allows to check open ports in defined range on host

### Using 
 `python scaner.py [start port] [end port] --host [ip, default=localhost]` 
 
 `python scaner.py --host 8.8.8.8 30 100`
 
 `python scaner.py 30 100`
